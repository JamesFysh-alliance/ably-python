ably-python
===========

Ably.io python client library

### Installation

#### From PyPi

    pip install ably-python

#### From a git url

    pip install git@bitbucket.org:jjwchoy/ably-python.git

#### Locally

    git clone git@bitbucket.org:jjwchoy/ably-python.git
    cd ably-python
    python setup.py install
